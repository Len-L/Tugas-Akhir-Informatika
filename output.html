Xmrig Install : Bitcoin : Unmineable Mining BTC menggunakan Xmrig CPU

 Sumber: https://xmrig.com/docs/miner/build/ubuntu   ==Deskripsi== * dicoba pada ubuntu 22.04 dan berhasil   ==Install xmrig di ubuntu== $> sudo apt-get install git build-essential cmake libuv1-dev libssl-dev libhwloc-dev $> git clone https://github.com/xmrig/xmrig.git  $> mkdir xmrig/build && cd xmrig/build $> cmake ..  $> make -j$(nproc)   ==Setup/Setting== Buat file dengan nama terserah dan isi file tersebut seperti berikut ini: cd xmrig/build ./xmrig -o rx.unmineable.com:3333 -a rx -k -u BTC:YourAddress.YourWorkerName Contoh: $> vim mulai-mining.sh cd /home/userrr/app/xmrig/build ./xmrig -o rx.unmineable.com:3333 -a rx -k -u BTC:123124weradsfSFDG34563SDFG546543.Userrr $> chmod +x mulai-mining.sh $> ./mulai-mining.sh selesai, sekarang laptop atau komputer anda sedang mining BTC ==Cek Hasil Mining Di Unmineable== Buka Browser >  Masuk Ke "https://www.unmineable.com/coins/BTC/address" > Masukan Alamat Dompet Digital Anda / Alamat Dompet
